O Mnie

Jako pasjonat programowania specjalizujący się w SQL i rozwoju backendu, dążę do budowania solidnych, wydajnych i skalowalnych aplikacji. Moja droga rozpoczęła się od solidnych podstaw informatyki, wspieranych praktycznym doświadczeniem oraz ciągłym doskonaleniem.

Interesuję się zarządzaniem bazami danych, analizą danych oraz technologiami backendowymi, a rozwiązywanie złożonych problemów za pomocą kodu to moja pasja. Mam doświadczenie w pracy z różnymi systemami baz danych oraz językami programowania.

Moje główne obszary zainteresowań to:

  • Projektowanie i optymalizacja baz danych
  • Pisanie zapytań SQL i optymalizacja wydajności
  • Tworzenie API RESTful
  • Znajomość metodologii Agile

Chciałbym pracować nad wymagającymi projektami, które pozwolą mi wykorzystać moje umiejętności techniczne i współpracować z utalentowanymi zespołami, aby tworzyć wpływowe rozwiązania programistyczne.

Umiejętności

  • SQL
  • Python
  • Java
  • Node.js
  • Git
  • REST APIs
